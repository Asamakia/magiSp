# マジックスピリット 大陸情報局（投資システム）

作成日: 2025年11月29日

## 概要

各地で起きる事件・異変・大会が、カード市場を動かす。
その「未来の変動」に賭ける投資システム。

```
表向き: 各地域の利権に投資するファンド
実態:   投資家同士の賭け（ブックメーカー方式）
情報局: 手数料で確実に儲かる
```

---

## 大陸情報局とは

各地に情報員を派遣し、情勢を収集・分析する組織。
投資家に「ファンド」を販売し、手数料で収益を得る。

### ビジネスモデル

```
投資家から資金を集める
　↓
「利権証書」として投資商品化
　↓
実際の利権もあるが、大半は投機マネー
　↓
投資家同士のお金が回る
　↓
情報局は手数料で確実に儲かる
```

→ **要するに「公認カジノ」**

## 投資の仕組み

### 購入から決済まで

```
1. ファンドを選ぶ
2. 口数を決めて購入（1口 = 1,000G〜）
3. 毎日維持コストが引かれる（0.2%/日）
4. 最大30日で強制決済
5. 好きなタイミングで売却可能
```

### コスト構造

| コスト | 金額 |
|--------|------|
| 購入手数料 | 2% |
| 維持コスト | 0.2%/日 |
| 売却手数料 | 2% |

### 30日フル保有時の例

```
投資額: 10,000G
購入手数料: 200G
維持コスト: 600G（0.2% × 30日）
売却手数料: 200G
合計コスト: 1,000G（10%）

→ 10%以上の値上がりで利益
```

| 要因 | 影響 |

```
　↓
🌙 黒呪の大陸ファンド +12%

事前に買ってた人が儲かる！
```

```

　↓
🔥 炎メタファンド +20%（的中ボーナス）
💧 水メタファンド -15%（外れペナルティ）
　↓
予想が当たった人が大儲け！
```

## AI対戦トーナメント賭け

### 概要

NPCたちがカードバトルのトーナメントを開催。
プレイヤーはどのNPCが勝つか賭ける。

```
NPCにデッキを持たせる
　↓
実際にAI同士でバトルさせる
　↓
結果は「本物」（運営も知らない）
　↓
プレイヤーはその結果に賭ける
```

---

### 大会の種類

| 大会 | 頻度 | 人数 | 最低賭け金 | 最大賭け金 |
|------|------|------|-----------|-----------|
| デイリーカップ | 毎日 | 4人 | 1,000G | 10,000G |
| ウィークリー杯 | 週1 | 8人 | 10,000G | 100,000G |
| マンスリー大会 | 月1 | 16人 | 50,000G | 500,000G |
| グランプリ | 季節 | 32人 | 100,000G | 2,000,000G |

---

### 賭け方の種類

| 賭け方 | 内容 | 配当目安 |
|--------|------|----------|
| 単勝 | 優勝者を当てる | オッズ通り |
| 複勝 | 上位半分に入るか | 低め（1.2〜2倍） |
| 決勝予想 | 決勝の2人を当てる | 高め（組み合わせ） |

---

### オッズの決まり方

```
基本オッズ = 過去戦績から算出
　↓
勝率高い → オッズ低い
勝率低い → オッズ高い
　↓
プレイヤーの賭けでさらに変動
```

## デッキ情報購入システム

### 基本ルール

```
大会エントリー時:
- NPCの名前と過去戦績は公開
- 今回のデッキ型は非公開

情報局で購入:
- 特定NPCのデッキ型を教えてもらえる
- 買わなければ勘で賭ける
```

NPCは同じテーマでも複数の構築パターンを持つ:

| NPC | メインテーマ | 構築パターン |
|-----|-------------|-------------|
| 炎帝カイン | 炎速攻 | A: 低コスト特化 / B: ドラゴン軸 / C: バーン混合 |
| 氷姫セイラ | 水コントロール | A: 除去重視 / B: 大型軸 / C: 回復耐久 |
| 聖騎士ロウェン | 光バランス | A: 鳥民軸 / B: 回復軸 / C: バフ軸 |
| 闘術師マリク | 闇バーン | A: 魔女軸 / B: 呪い軸 / C: 墓地利用 |
| 獣王ガロン | 原始ビート | A: 粘液獣軸 / B: 岩狸軸 / C: 大型軸 |
| 機工士ノヴァ | 未来コンボ | A: ゴーレム軸 / B: 未来鴉軸 / C: 展開軸 |
| 謎の仮面 | 混合 | A: 禁忌軸 / B: 混合ビート / C: 奇襲型 |
| 新人トーマ | 混合 | A: 初心者構築 / B: 借り物デッキ / C: 運任せ |

---

### 情報の種類と価格

| 情報 | 内容 |
|------|------|
| デッキ型 | 「カインは今回ドラゴン軸」 |
| キーカード | 「フレア・ドラゴンSR採用」（2〜3枚公開） |
| フルリスト | デッキ全容公開 |

大会規模で価格変動:

| 大会 | デッキ型 | キーカード | フルリスト |
|------|---------|-----------|-----------|
| デイリー | 500G | 1,500G | 5,000G |
| ウィークリー | 1,000G | 3,000G | 10,000G |
| マンスリー | 3,000G | 10,000G | 30,000G |
| グランプリ | 10,000G | 30,000G | 100,000G |

---

### 情報の価値

```
情報なしで賭ける:
　→ 運任せ、でも情報代は浮く

情報買って賭ける:
　→ 勝率上がる、でもコストかかる

どっちが得か？の駆け引き
```

戦略パターン:

| 戦略 | メリット | デメリット |
|------|----------|-----------|
| 全員の情報買う | 精度最大 | コスト大、利益減 |
| 本命だけ買う | バランス | 穴は読めない |
| 誰も買わない | 利益最大化 | 完全に運任せ |
