# マジックスピリット 商人NPCシステム

作成日: 2025年11月28日

> **⚠️ 実装状況: 未実装（仕様書のみ）**
>
> このドキュメントは将来実装予定の商人NPCシステムの仕様書です。
> 現在はパック購入とコレクション画面からの売却のみが実装されています。
> 関連ドキュメント: `collection-system-design.md`

## 概要

シングルカードの売買は商人NPCを通じて行う。
商人ごとに専門分野があり、品揃えや買取価格が異なる。

```
パック → ランダムでカード入手
商人   → 狙ったカードを購入/売却
```

---

## 商人の種類

| 種類 | 人数 | 特徴 |
|------|------|------|
| 属性商人 | 6人 | 各属性専門、曜日制（日曜は全員） |
| 一般商人 | 1人 | 全属性、常設 |
| 闇商人 | 1人 | 禁忌・高レア、ランダム出現 |
| 旅商人 | 1人 | 掘り出し物、低確率出現 |
| コレクター | 6人 | カテゴリ専門、日替わり出現 |

---

## 価格設定

| 取引種類 | 価格 |
|----------|------|
| 購入価格 | 市場価値 × 1.0〜1.3 |
| 売却価格 | 市場価値 × 0.7〜0.9 |
| 闇商人 | 市場価値 × 1.2〜1.5（高いがレア） |
| コレクター買取 | 専門カテゴリは × 1.2（高く買ってくれる） |

## 噂話システム

### 概要

商人に話しかけると、一定確率で噂話をしてくれる。
翌日以降の市場変動のヒントになる。

| 項目 | 内容 |
|------|------|
| 発生確率 | 話しかけるたび30%程度 |
| 更新 | 1日1回（同じ日は同じ噂） |
| 的中率 | 70〜80%（たまに外れる） |
| 優先度 | 低（後回しでOK） |

### 仕組み

```
次週のデイリーニュース・突発イベントを先に決定
　↓
商人が「それを匂わせる噂」を話す
　↓
プレイヤーは噂を元に売買判断
　↓
たまに外れる（ガセネタ20〜30%）
```

### 噂の種類

| 種類 | 内容 | ヒント先 |
|------|------|----------|
| 属性の噂 | 「炎が熱い」「水が冷え込む」 | 属性変動 |
| カテゴリの噂 | 「ドラゴンが注目されてる」 | カテゴリ変動 |
| 商人出現の噂 | 「怪しい商人を見た」 | 「名無し」出現 |
| イベントの噂 | 「嵐が来るらしい」 | 突発イベント |
| ガセネタ | 上記と同じだが外れる | なし |

### キャラ別の噂傾向

| 商人 | 得意な噂 | 口調例 |
|------|----------|--------|
| ボルガン | 炎系、大雑把 | 「ドラゴンが来るぜ、たぶんな！」 |
| クラディア | 水系、駆け引き | 「水属性、仕込んでおくといいかも？」 |
| パスト神父 | 光系、道徳的 | 「光の時代が来ると信じております」 |
| ダスカ | 闇系、不吉 | 「闇が騒いでるねぇ...何か来るよ」 |
| ゴブス | 原始系、自然 | 「沼地がなんか変だなあ」 |
| ドクトル・ギア | 未来系、分析 | 「データ上、ゴーレム需要が上昇傾向なのだよ」 |
| マルクス | 全般、無難 | 「最近〇〇がよく売れてますよ」 |
| コレクター | 専門カテゴリ | 「絶対来る！俺には分かる！」（ピッポ） |

## 依頼システム

### 概要

コレクターが特定カードの収集依頼を出す。
達成すると報酬がもらえる。

| 項目 | 内容 |
|------|------|
| 発行者 | コレクター6人 |
| 依頼数 | 1人1件（同時進行可能） |
| 期限 | 7日（7戦）or 無期限 |
| 更新 | 達成or期限切れで新しい依頼 |

### 依頼の難易度

| 難易度 | 内容例 | 報酬 |
|--------|--------|------|
| ★☆☆ | 「〇〇を3枚集めて」 | 1,000G〜3,000G |
| ★★☆ | 「〇〇のR以上を2枚」 | 5,000G〜10,000G |
| ★★★ | 「〇〇のSR以上を1枚」 | 15,000G〜 + 特別報酬 |

### 特別報酬

| 報酬タイプ | 内容 |
|------------|------|
| 限定スリーブ | コレクターデザイン |
| 称号 | 「岩狸マスター」「リリカの友」など |
| カードフレーム | デッキ画面で映える装飾 |
| 特別カード | サイン入り版（性能同じ、見た目違い） |
| 商人割引券 | 次回そのコレクターから10%オフ |
| 情報 | 次週の市場ヒント確定版 |

### コレクター別の依頼・報酬

| コレクター | 依頼傾向 | 特別報酬 |
|------------|----------|----------|
| ピッポ | 岩狸/粘液獣の数集め | 「岩狸マニア」称号、岩狸スリーブ |
| リリアーヌ | リリカ/魔女のレア指定 | リリカフレーム、ピンクスリーブ |

## トレードシステム

### 概要

旅商人ゼルヴァンがカード交換を持ちかけてくる。
お金を介さないカード同士のトレード。目利き勝負。

| 項目 | 内容 |
|------|------|
| 発行者 | ゼルヴァン（出現時のみ） |
| 出現率 | 5%程度 |
| 交換数 | 出現ごとに3件程度 |
| 内容 | ランダム（お得も損もある） |

### 交換の種類

| 種類 | 内容 | 例 |
|------|------|-----|
| 等価交換 | 同程度の価値同士 | 「岩狸R ↔ 粘液獣R」 |
| お得交換 | こちらが得 | 「C3枚 → R1枚」 |
| レア交換 | 入手困難なものが出る | 「SR2枚 → 限定カード」 |
| 変わり種 | 属性違いなど | 「炎R3枚 → 水SR1枚」 |
| ギャンブル | 結果が見えない | 「禁忌1枚 → ？？？」 |

### 目利き勝負の面白さ

```
市場価格を知っていれば得できる
　↓
知らないと損することも
　↓
レアな交換は1回限り（早い者勝ち）
```

