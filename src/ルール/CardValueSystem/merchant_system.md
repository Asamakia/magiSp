## 概要

シングルカードの売買は商人NPCを通じて行う。
商人ごとに専門分野があり、品揃えや買取価格が異なる。

```
パック → ランダムでカード入手
商人   → 狙ったカードを購入/売却
```

---

## 商人の種類

| 種類 | 人数 | 特徴 |
|------|------|------|
| 属性商人 | 6人 | 各属性専門、曜日制（日曜は全員） |
| 一般商人 | 1人 | 全属性、常設 |
| 闇商人 | 1人 | 禁忌・高レア、ランダム出現 |
| 旅商人 | 1人 | 掘り出し物、低確率出現 |
| コレクター | 6人 | カテゴリ専門、日替わり出現 |

## 価格設計

### 販売価格（プレイヤーが買う）

```
販売価格 = 市場価格 × 商人倍率 × (1 - 好感度割引)
```

| 商人 | 基本倍率 | 専門補正 | 説明 |
|------|----------|----------|------|
| マルクス | ×1.00 | - | 標準価格（基準） |
| 属性商人 | ×1.05 | 専門属性: ×1.00 | 専門外は少し高い |
| コレクター | ×1.05 | 専門カテゴリ: ×0.95 | 専門は安く売ってくれる |
| 闇商人 | ×1.30 | 禁忌: ×1.50 | 高いが確実 |
| 旅商人 | ×0.50〜0.80 | ランダム | 掘り出し物 |

### 買取価格（プレイヤーが売る）

```
買取価格 = 市場価格 × 商人倍率 × (1 + 好感度ボーナス)
```

| 商人 | 基本倍率 | 専門補正 | 説明 |
|------|----------|----------|------|
| マルクス | ×0.70 | - | 標準（7掛け） |
| 属性商人 | ×0.75 | 専門属性: ×0.85 | 専門は高く買う |
| コレクター | ×0.70 | 専門カテゴリ: ×1.00 | **専門は等価買取！** |
| 闇商人 | ×0.60 | 禁忌: ×0.80 | 高レアは買い叩く |
| 旅商人 | - | - | **買取なし（トレードのみ）** |

### 価格例（市場価格1,000Gのカード）

| 商人 | 販売価格 | 買取価格 | 備考 |
|------|----------|----------|------|
| マルクス | 1,000G | 700G | 安定 |
| 属性商人（専門） | 1,000G | 850G | 専門は優遇 |
| 属性商人（専門外） | 1,050G | 750G | 専門外は普通 |
| コレクター（専門） | 950G | **1,000G** | **等価買取！** |
| コレクター（専門外） | 1,050G | 700G | 専門外は普通 |
| 闇商人 | 1,300G | 600G | 売買スプレッド大 |
| 旅商人 | 500〜800G | - | 買取なし |

## 相場認識ラグ

商人ごとに市場価格の認識にラグがある。
これによりアービトラージ（価格差取引）の機会が生まれる。

| 商人タイプ | ラグ | 説明 |
|------------|------|------|
| マルクス（一般） | 0日 | リアルタイム追従 |
| 属性商人 | 2日 | 2戦前の市場価格で取引 |
| コレクター | 専門0日 / 専門外5日 | 専門は即時、他は鈍い |
| 闇商人 | 1日 | ほぼリアルタイム（情報通） |
| 旅商人 | 3日 | 各地を巡っているので遅れる |

### アービトラージ例

```
【例: 属性商人のラグ利用】
1. 今日: 水属性が暴騰（+50%）、市場価格 1,500G
2. クラディア（水商人、2日ラグ）: まだ 1,000G で販売
3. クラディアから買って、マルクスに売る
→ 1,000G で買って 1,050G（1,500×0.7）で売れる
```

## 好感度システム

商人との取引で好感度が上昇し、様々なボーナスを得られる。

### 好感度レベル

| Lv | 必要取引回数 | 販売割引 | 買取ボーナス | 追加効果 |
|----|-------------|----------|--------------|----------|
| 1 | 0回 | 0% | 0% | なし |
| 2 | 5回 | 5% | 0% | - |
| 3 | 15回 | 10% | 5% | - |
| 4 | 30回 | 15% | 10% | 品揃え+2枠 |
| 5 | 50回 | 20% | 15% | 噂的中率+10% |

### 取引カウント

- 購入1回 = +1
- 売却1回 = +1
- 依頼達成 = +5（コレクターのみ）

### 好感度適用後の価格例（Lv5、市場価格1,000G）

| 商人 | 販売価格 | 買取価格 |
|------|----------|----------|
| マルクス | 800G | 805G |
| 属性商人（専門） | 800G | 977G |
| コレクター（専門） | 760G | **1,150G** |

## 在庫・仕入れシステム

### 在庫の流れ

```
┌──────────────┐
│ カードプール │ ← 433枚から抽出
└──────┬───────┘
       ↓ 毎戦リセット
┌──────────────┐
│ 商人の品揃え │ ← 5〜10枚
└──────┬───────┘
       ↓ プレイヤーが購入
┌──────────────┐
│ 在庫から消える│
└──────────────┘
```

### プレイヤー売却時の在庫化

プレイヤーがカードを売却すると、一定確率で商人の在庫になる。

| 商人タイプ | 在庫化確率 | 出現タイミング |
|------------|------------|----------------|
| マルクス | 80% | 翌戦 |
| 属性商人 | 60% | 1〜2戦後 |
| コレクター | 100%（専門のみ） | 翌戦 |
| 闇商人 | 30% | 出現時に判定 |
| 旅商人 | 50% | 出現時に判定 |

### 在庫システムの詳細

- 各商人: 最大20枚（古いものから消える）
- プレイヤー売却品には「売却者マーク」（自分が売ったと分かる）
- レアカードを売ると、後日その商人から買い戻せる可能性
- コレクターに専門カードを売ると確実に在庫化

## 商人呼び出しシステム

### 呼び出しチケット

| チケット種類 | 価格 | 効果 |
|--------------|------|------|
| 属性商人呼び出し券 | 500G | 曜日外の属性商人を1戦だけ呼ぶ |
| 闇商人の招待状 | 2,000G | 「名無し」を確定出現 |
| 旅商人の風笛 | 1,500G | ゼルヴァンを確定出現 |

### 入手方法

- ショップで購入（常時販売）
- 依頼報酬（コレクターから）
- 対戦報酬（低確率）

### コレクター確定出現条件

専門カテゴリを5枚以上売却すると、翌日そのコレクターが確定出現。

例: 岩狸を5枚売る → 翌日ピッポ確定出現

## ウィッシュリスト

プレイヤーが「欲しいカード」を最大10枚登録できる。
商人が該当カードを持っていると通知される。

### 機能

- コレクション画面から登録
- カード詳細画面に「欲しいものリストに追加」ボタン
- 商人ギルド画面で該当商人に⭐マーク表示
- 「おすすめ！」通知でハイライト

## 噂話システム

### 概要

商人に話しかけると、一定確率で噂話をしてくれる。
翌日以降の市場変動のヒントになる。

| 項目 | 内容 |
|------|------|
| 発生確率 | 話しかけるたび30%程度 |
| 更新 | 1日1回（同じ日は同じ噂） |
| 的中率 | 70〜80%（たまに外れる） |
| 優先度 | 低（後回しでOK） |

### 仕組み

```
次週のデイリーニュース・突発イベントを先に決定
　↓
商人が「それを匂わせる噂」を話す
　↓
プレイヤーは噂を元に売買判断
　↓
たまに外れる（ガセネタ20〜30%）
```

### 噂の種類

| 種類 | 内容 | ヒント先 |
|------|------|----------|
| 属性の噂 | 「炎が熱い」「水が冷え込む」 | 属性変動 |
| カテゴリの噂 | 「ドラゴンが注目されてる」 | カテゴリ変動 |
| 商人出現の噂 | 「怪しい商人を見た」 | 「名無し」出現 |
| イベントの噂 | 「嵐が来るらしい」 | 突発イベント |
| ガセネタ | 上記と同じだが外れる | なし |

### キャラ別の噂傾向

| 商人 | 得意な噂 | 口調例 |
|------|----------|--------|
| ボルガン | 炎系、大雑把 | 「ドラゴンが来るぜ、たぶんな！」 |
| クラディア | 水系、駆け引き | 「水属性、仕込んでおくといいかも？」 |
| パスト神父 | 光系、道徳的 | 「光の時代が来ると信じております」 |
| ダスカ | 闘系、不吉 | 「闇が騒いでるねぇ...何か来るよ」 |
| ゴブス | 原始系、自然 | 「沼地がなんか変だなあ」 |
| ドクトル・ギア | 未来系、分析 | 「データ上、ゴーレム需要が上昇傾向なのだよ」 |
| マルクス | 全般、無難 | 「最近〇〇がよく売れてますよ」 |
| コレクター | 専門カテゴリ | 「絶対来る！俺には分かる！」（ピッポ） |

## 依頼システム（後日実装）

### 概要

コレクターが特定カードの収集依頼を出す。
達成すると報酬がもらえる。

| 項目 | 内容 |
|------|------|
| 発行者 | コレクター6人 |
| 依頼数 | 1人1件（同時進行可能） |
| 期限 | 7日（7戦）or 無期限 |
| 更新 | 達成or期限切れで新しい依頼 |

### 依頼の難易度

| 難易度 | 内容例 | 報酬 |
|--------|--------|------|
| ★☆☆ | 「〇〇を3枚集めて」 | 1,000G〜3,000G |
| ★★☆ | 「〇〇のR以上を2枚」 | 5,000G〜10,000G |
| ★★★ | 「〇〇のSR以上を1枚」 | 15,000G〜 + 特別報酬 |

### 特別報酬

| 報酬タイプ | 内容 |
|------------|------|
| 限定スリーブ | コレクターデザイン |
| 称号 | 「岩狸マスター」「リリカの友」など |
| カードフレーム | デッキ画面で映える装飾 |
| 特別カード | サイン入り版（性能同じ、見た目違い） |
| 商人割引券 | 次回そのコレクターから10%オフ |
| 情報 | 次週の市場ヒント確定版 |

## トレードシステム（後日実装）

### 概要

旅商人ゼルヴァンがカード交換を持ちかけてくる。
お金を介さないカード同士のトレード。目利き勝負。

| 項目 | 内容 |
|------|------|
| 発行者 | ゼルヴァン（出現時のみ） |
| 出現率 | 5%程度 |
| 交換数 | 出現ごとに3件程度 |
| 内容 | ランダム（お得も損もある） |

### 交換の種類

| 種類 | 内容 | 例 |
|------|------|-----|
| 等価交換 | 同程度の価値同士 | 「岩狸R ↔ 粘液獣R」 |
| お得交換 | こちらが得 | 「C3枚 → R1枚」 |
| レア交換 | 入手困難なものが出る | 「SR2枚 → 限定カード」 |
| 変わり種 | 属性違いなど | 「炎R3枚 → 水SR1枚」 |
| ギャンブル | 結果が見えない | 「禁忌1枚 → ？？？」 |

### リスク表示オプション

- 初心者向け: 推定価値比較を表示（市場分析画面連動）
- 上級者向け: 非表示オプションで目利き勝負
